# node-static

Static file server

## Requirements

- Node

## Usage

```console
$ npx @aminnairi/serve help
```

## Example

```console
$ npx @aminnairi/serve from public
Serving from /home/user/project/public on http://0.0.0.0:8080 (CTRL-C to stop gracefully)
```
